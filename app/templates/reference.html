üß† Tips for Devs

  • Restart Odoo after code changes: sudo service odoo restart
  • Use tail -f /var/log/odoo/odoo.log to debug in real time
  • Edit /etc/odoo/odoo.conf to change ports, paths, etc.
  • Extend addons_path to include custom addons:
addons_path = /odoo/custom/addons,/usr/lib/python3/dist-packages/odoo/addons
